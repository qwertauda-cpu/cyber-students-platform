# رفع التغييرات إلى GitHub

## الخطوات السريعة:

1. **افتح GitHub وأنشئ المستودع:**
   - اذهب إلى: https://github.com/new
   - اسم المستودع: `cyber-students-platform`
   - اختر Public أو Private
   - **لا تضع** README أو .gitignore أو license
   - اضغط "Create repository"

2. **بعد الإنشاء، نفذ هذا الأمر:**
   ```powershell
   git push -u origin main
   ```

## أو استخدم هذا الأمر المباشر:

```powershell
git push origin main
```

---

**ملاحظة:** التغييرات محفوظة محلياً وجاهزة للرفع. فقط أنشئ المستودع على GitHub ثم ارفعها.
